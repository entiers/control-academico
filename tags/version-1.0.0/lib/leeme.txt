Nombre que se les debe de colocar a las librerías cuando se agregan a Netbeans

	- Jasypt
	- SpringSecurity
	- HibernateValidation
	- Log4j
	- Sitemesh
	- DWR
	- JodaTime
	- PostgreSQL-JDBC4

El nombre de las librerías "Spring Web MVC 2.5" y "Spring Framework 2.5" sigue
siendo el mismo, solo se deben de sustituir los archivos "jar" por los de la
versión 3.

Se debe de cambiar la libreria "PostgreSQL JDBC Driver" que trae Netbeans por la
libreria "PostgreSQL-JDBC4", esto es porque la version del driver jdbc que trae
Netbeans no es compatible con la version 1.6 del JDK.
